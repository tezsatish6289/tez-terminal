# **App Name**: TezTerminal Antigravity

## Core Features:

- Webhook Ingestion: Receive trading signals from TradingView webhooks.
- Webhook History Display: Display a history of received webhook alerts in a web page.

## Style Guidelines:

- Primary color: Dark desaturated purple (#38304A) for a professional yet accessible feel.
- Background color: Very dark desaturated purple (#13111A) to maintain focus on the trading data.
- Accent color: Electric cyan (#7DF9FF) for highlighting key information and calls to action.
- Body and headline font: 'Inter', a sans-serif font with a modern, neutral, machined look.
- Use clear, minimalistic icons to represent different signal types and indicators.
- A clear layout to help Lucknow traders with quick navigation. Left sidebar for watchlist, main pane for TradingView charts, and top bar for account and settings.
- Subtle transitions and animations for a smooth user experience, with no distracting or unnecessary animations.
